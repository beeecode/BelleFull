# Amazing Taste Delicacies Brand Palette

Approximate palette extracted from the provided Instagram flyer.

## Core Colors

- Food orange: `#F15A1B`
- Fresh green: `#24B52D`
- Deep green: `#138F22`
- Rice gold: `#D97617`
- Pepper red: `#D72513`
- Charcoal: `#2B2522`
- Warm cream: `#FFF5ED`
- White: `#FFFFFF`

## Usage

- Use food orange for brand warmth, large accent panels, and logo-adjacent moments.
- Use fresh green for primary CTAs, active states, badges, and "order" actions.
- Use warm cream as the soft page background where white feels too plain.
- Use charcoal for strong text and dark UI details.
- Use pepper red sparingly for small food accents only.
